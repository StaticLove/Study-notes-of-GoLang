### The Tips of MAC OS

#### The Command of MAC

	终端清屏(清除屏幕缓存)： Command + K
	终端清屏(不清除屏幕缓存)： Command + R

    关闭屏幕（黑屏）：Ctrl + Shift + Power
    睡眠(sleep)：Cmd + Opt + Power
    重启(restart)： Cmd + Ctrl + Power
    关机(shutdown)：Cmd + Ctrl + Opt + Power

#### The Command of Shell
    
    shell 的 preference：Cmd + i
